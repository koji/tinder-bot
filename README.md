# tinder-bot


## requirements
Facebook account  

## how to use
* 1 install webdriver
```zsh
$ brew cask install chromedriver
# check version
$ chromedriver --version
```

* 2 run python script
```zsh
$ git clone https://github.com/koji/tinder-bot.git
$ cd tinder-bot
$ pip install -r requirements.txt
```
